# How to install
## On Debian/Ubuntu

```
sudo apt-get install php5-dev build-essential git
git clone https://github.com/miyandi1997/DECODEP
cd DECODEP
phpize
./configure
make
sudo make install
```

# How to use

```
php -d extension=decodep.so bot.php
```
