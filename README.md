# How to install
## On Debian/Ubuntu

```
su root
apt-get install php-dev 
apt-get install build-essential 
apt-get install git
git clone https://github.com/miyandi1997/DECODEP
cd DECODEP
phpize
./configure
make
sudo make install
```

# How to use

```
php -d extension=decodep.so bot.php
```
